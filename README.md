# Wuthering Waves AES Keys

## Usage

To use these AES keys in [FModel](https://fmodel.app/), configure the **AES Endpoint** in your settings:

- **Endpoint:** https://yarik0chka.github.io/wuwa-keys/keys.json
- **Expression:** `$['mainKey', 'dynamicKeys']`

1. Go to **Settings → Endpoint Configuration → AES**.
2. Input values from above.
3. Click **Test**, then **OK**.
4. Restart FModel or refresh the keys in **Directory → AES** window.
